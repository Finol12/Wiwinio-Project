### Folder structure

The project folder is organized as follows:

```
.
└── Repository Name/
    ├── data/
    │   └── < data files (e.g. db, csv, pickles, etc.) >
    ├── src/
    │   └── < source files >
    ├── notebooks/
    │   └── < notebooks for data exploration >
    ├── output/
    │   └── < any artifacts of analysis (graphs, model objects, etc.) >
    ├── .gitattributes
    ├── .gitignore
    ├── requirements.txt
    ├── main.py
    └── README.md

```
